# wpggLabs

**hobbyist · tinkerer · builds in the open**

i make **local-first** software — tools that run on *your* machine, keep *your* data, and answer to *you*. no accounts, no telemetry, no cloud landlord. if it can work offline, it should.

i build in the **open** because software should be something you can read, fork, and actually own — not rented from whoever's holding your data hostage this quarter. small independent tools beat another walled garden.

**what i'm doing now**
scratching my own itches — desktop apps, browser tools, and bits for streamers. nights, weekends, and the occasional 3am *"just one fix."* some ship, some rot in a branch. it's a hobby, not a startup, and i'd like to keep it that way.
