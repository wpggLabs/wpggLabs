# wpggLabs

**hobbyist · tinkerer · builds in the open**

i make things because i want them to exist — then open them up so anyone can read, fork, or run them. no gatekeeping, no walled gardens. small independent tools you can actually own beat another product you just rent.

**what i'm doing now**
open-sourcing more of what i build — ideas, experiments, half-finished tools and all. i've been sitting on too many of them; figured they're more useful in the open than rotting in a private repo. a hobby, not a startup.

a chronic idea-starter, compulsive finisher: adhd picks the projects, ocd won't let them ship half-done — too many ideas, impossibly high standards for the few that survive.
